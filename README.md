# Cpe112 - Course Prerquisite Checker

The Course Prerequisite Checker System is a C programming project that helps students check whether they meet the requirements for enrolling in a course. The program stores courses, prerequisite subjects, and minimum required grades, then compares them with the student’s academic profile. It uses structures, arrays, linked lists, recursion, and dynamic memory allocation to manage and display prerequisite relationships. The system also includes a menu-driven interface that allows users to add completed courses, update grades, and check prerequisite status easily.


How to run:
1. Run the program
2. Select your choice
3. Follow the instruction (Typing in courses name and grade if you are adding them)
